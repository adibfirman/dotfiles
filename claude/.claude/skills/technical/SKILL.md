# Technical Skill Router

Use this skill for broad technical software engineering work.

## Delegate to child skill groups

Use `react` for:
- React
- Next.js
- React component architecture
- React performance
- React Native

Use `golang` for:
- Go code
- Go architecture
- Go tooling
- Go testing
- Go performance
- Go libraries
- Go debugging

## Default behavior

Identify whether the task is React-related or Go-related.
Load the matching child skill before planning or implementation.
If neither React nor Go applies, do not force this skill.
