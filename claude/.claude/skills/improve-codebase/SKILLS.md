# Improve Codebase Skills

Read this folder as a hierarchy:

1. Start with `codebase-design/` for the shared architecture vocabulary: module, interface, depth, seam, adapter, leverage, locality.
2. Then use `improve-codebase-architecture/` to scan a codebase for deepening opportunities.

`improve-codebase-architecture` depends on the vocabulary from `codebase-design`.
